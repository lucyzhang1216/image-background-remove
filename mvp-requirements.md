# MVP 需求文档：图片背景移除工具

**项目代号**: Image Background Remover  
**版本**: v1.0  
**日期**: 2026-03-28  

---

## 1. 项目概述

### 背景
用户需要一个简单高效的在线工具，快速移除图片背景。

### 目标
提供一个 minimal viable product (MVP)，让用户通过网页上传图片，自动获得去除背景的透明图。

---

## 2. 功能范围

### 核心功能 (MVP)

| 优先级 | 功能 | 描述 |
|--------|------|------|
| P0 | 图片上传 | 支持拖拽或点击上传，支持 JPG/PNG |
| P0 | 背景移除 | 调用 Remove.bg API 处理图片 |
| P0 | 结果展示 | 页面显示处理后的透明背景图 |
| P0 | 图片下载 | 一键下载去除背景后的图片 |
| P1 | 进度指示 | 上传/处理中显示 loading 状态 |
| P1 | 错误处理 | API 失败时提示用户重试 |

### 非功能范围 (MVP 暂不包含)

- 用户登录/注册
- 历史记录
- 批量处理
- 付费订阅

---

## 3. 技术方案

### 架构图

```
┌─────────────┐     ┌──────────────────┐     ┌─────────────────┐
│   前端页面   │ ──▶ │  Cloudflare Worker │ ──▶ │   Remove.bg API  │
│  (静态托管)  │     │    (后端服务)      │     │   (图片处理)      │
└─────────────┘     └──────────────────┘     └─────────────────┘
```

### 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | HTML + Vanilla JS | 简单轻量，无需框架 |
| 后端 | Cloudflare Workers | 免费额度，无服务器运维 |
| 图片处理 | Remove.bg API | 成熟方案，效果稳定 |
| 部署 | Cloudflare Pages | 静态托管前端 |

### API 配置

```
Remove.bg API Key: (用户提供)
Endpoint: https://api.remove.bg/v1.0/remove-background
```

## 4. 页面设计

### 页面结构

```
┌─────────────────────────────────────┐
│            Header                   │
│         "背景移除工具"               │
├─────────────────────────────────────┤
│                                     │
│         [上传区域]                   │
│      点击或拖拽上传图片              │
│                                     │
├─────────────────────────────────────┤
│         [处理结果]                   │
│    ┌─────────┐    ┌─────────┐      │
│    │ 处理前  │ →  │ 处理后  │      │
│    └─────────┘    └─────────┘      │
│                                     │
│         [下载按钮]                   │
│           下载图片                   │
│                                     │
└─────────────────────────────────────┘
```

### 交互流程

1. 用户进入页面
2. 点击上传区域或拖拽图片
3. 显示"处理中..."状态
4. 完成后展示原图 vs 结果图
5. 点击下载按钮保存

## 5. 接口设计

### Worker 入口

```
POST /api/remove-bg
Content-Type: multipart/form-data

请求体: image (图片文件)

响应: 
- 成功: 返回透明 PNG 图片 (image/png)
- 失败: 返回 JSON { error: "错误描述" }
```

### 前端调用

```javascript
const formData = new FormData();
formData.append('image', file);

fetch('/api/remove-bg', {
  method: 'POST',
  body: formData
})
```
